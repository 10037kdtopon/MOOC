PREGUNTAS

Pregunta 1

¿Cuál es la principal diferencia entre los lenguajes C y C++ y qué ventajas ofrecen para el desarrollo de sistemas operativos y software de aplicaciones?

Respuesta

El lenguaje C es un lenguaje de programación estructurada con eficiencia de bajo nivel, lo que permite manipular directamente la memoria. Por esta razón, se utiliza en el desarrollo de sistemas operativos como Windows, Mac y Linux, así como en software de control industrial y bases de datos. El lenguaje C++, creado por Bjarne Stroustrup en 1979, es un lenguaje híbrido que combina programación estructurada y orientada a objetos, permitiendo la manipulación de objetos y abstracciones. Esto lo hace adecuado tanto para sistemas de bajo nivel como para aplicaciones complejas, matemáticas y de escritorio.

Pregunta 2

¿Qué tipos de comentarios existen en C/C++ y por qué son importantes?

Respuesta

Existen comentarios de una sola línea (//) y de varias líneas (/* … */). Son útiles para explicar el código y facilitar su comprensión.

Pregunta 3

¿Qué herramienta se utiliza para crear programas en C/C++ y qué ventajas ofrece?

Respuesta

Visual C++ .NET, parte de Visual Studio Community, permite crear, compilar y ejecutar programas en C/C++ de forma sencilla. Ofrece un entorno visual amigable, gratuito y compatible con las versiones 2019 y 2022 de Windows.

Pregunta 4

¿Qué diferencia hay entre cout y printf() al mostrar mensajes?

Respuesta

cout pertenece a C++ y utiliza el operador <<, mientras que printf() es de C y usa cadenas de formato con caracteres de escape como \n. Ambas permiten mostrar texto, pero cout es más moderno y orientado a objetos.
