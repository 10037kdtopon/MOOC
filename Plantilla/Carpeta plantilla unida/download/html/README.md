# AppStar! Template - Proyecto Organizado

## 📁 Estructura del Proyecto

```
html/
├── index3.html              # Archivo principal de la plantilla
├── index2.html              # Archivo conservado (con screenshots originales)
├── test-slider.html         # Archivo de prueba del slider
├── README.md               # Este archivo
│
├── stylesheets/            # Archivos CSS
│   ├── base.css           # Estilos base
│   ├── skeleton.css       # Grid system
│   ├── layout1.css        # Layout principal (verde turquesa)
│   ├── magnific-popup.css # Estilos para popups
│   └── screenshots-slider.css # Estilos del slider de screenshots
│
├── js/                     # Archivos JavaScript
│   ├── jquery-1.8.2.min.js # jQuery
│   ├── modernizr.custom.js  # Modernizr
│   ├── scripts.js          # Scripts principales
│   ├── screen.js           # Scripts de pantalla
│   ├── fitvids.js          # Videos responsivos
│   ├── jquery.sidr.min.js  # Menú lateral
│   ├── jquery.magnific-popup.min.js # Popups
│   └── screenshots-slider.js # JavaScript del slider
│
├── assets/                 # Recursos
│   ├── flexslider/        # Slider principal
│   └── fonts/             # Fuentes (Font Awesome)
│
└── images/                # Imágenes
    ├── backgrounds/       # Fondos
    ├── slider/           # Imágenes del slider principal
    ├── screenshots/      # Screenshots
    ├── team/            # Fotos del equipo
    └── [otros archivos de imagen]
```

## 🎨 Características

### Slider de Screenshots
- **Archivo CSS**: `stylesheets/screenshots-slider.css`
- **Archivo JS**: `js/screenshots-slider.js`
- **Funcionalidades**:
  - Navegación con flechas
  - Indicadores de puntos
  - Auto-play (5 segundos)
  - Soporte para touch/swipe
  - Navegación con teclado
  - Responsive design

### Colores del Tema
- **Verde turquesa principal**: `#1abc9c`
- **Verde oscuro**: `#16a085`
- **Gris oscuro**: `#333`
- **Blanco**: `#fff`

## 🚀 Uso

1. **Archivo principal**: `index3.html`
2. **Para desarrollo**: `index2.html` (conservado)
3. **Para pruebas**: `test-slider.html`

## 📝 Notas

- Los archivos no utilizados han sido eliminados
- El proyecto está optimizado y organizado
- Todos los estilos y scripts están en archivos separados
- Compatible con navegadores modernos
- Responsive design incluido

## 🔧 Personalización

Para cambiar colores, edita `stylesheets/layout1.css`
Para modificar el slider, edita `stylesheets/screenshots-slider.css` y `js/screenshots-slider.js`
